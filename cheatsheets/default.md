Hammerspoon Cheatsheet
======================

Hyper = CapsLock (via Karabiner)

## Window: Directional Tiling

| Key | Action |
| ---: | --- |
| Hyper-← | Tile left (1/2 → 1/3 → 1/4 → 1/6) |
| Hyper-→ | Tile right (1/2 → 1/3 → 1/4 → 1/6) |
| Hyper-↑ | Tile top |
| Hyper-↓ | Tile bottom |
| Hyper-F | Fullscreen (full → 3/4 → 1/2 → 1/3) |
| Hyper-M | Center window (keep size) |
| Hyper-N | Move to next screen |
| Hyper-Z | Maximize |

## Window: Column Zones

| Key | Action |
| ---: | --- |
| Hyper-1 | Left 1/3 (→ 1/4 → 1/6) |
| Hyper-2 | Center 1/3 (→ 1/4) |
| Hyper-3 | Right 1/3 (→ 1/4 → 1/6) |
| Hyper-4 | Left 2/3 (→ 3/4) |
| Hyper-5 | Right 2/3 (→ 3/4) |
| Hyper-6 | Center cycle (1/3 → 1/2 → 2/3) |

## Window: Quadrants

| Key | Action |
| ---: | --- |
| Hyper-Q | Top-left (1/2 → 1/3 → 1/4) |
| Hyper-W | Top-right (1/2 → 1/3 → 1/4) |
| Hyper-A | Bottom-left (1/2 → 1/3 → 1/4) |
| Hyper-D | Bottom-right (1/2 → 1/3 → 1/4) |

## Window: 2×3 Grid

| | Left | Center | Right |
| ---: | :---: | :---: | :---: |
| Top | Hyper-U | Hyper-I | Hyper-O |
| Bottom | Hyper-7 | Hyper-8 | Hyper-9 |

Edge cells cycle 1/3 → 1/4, center cells cycle 1/3 → 1/2.

## Window: Nudge

| Key | Action |
| ---: | --- |
| Hyper-- | Slide window left |
| Hyper-= | Slide window right |
